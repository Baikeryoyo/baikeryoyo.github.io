流程控制

在一个程序执行的过程中，代码块执行的顺序是对程序运行结果有直接的影响

在PHP中，使用多种流程控制语句，以便使相应的代码在代码块中执行不同的操作结果

流程控制代替语法

PHP 提供了一些流程控制的代替语法，包括if，while，for，foreach，switch

代替语法的基本形式是把花括号进行替换

左花括号替换为:，右花括号替换为endif，endwhile，endfor，endforeach，endswitch

if 条件控制语句

(PHP 4, PHP 5, PHP 7, PHP 8)

if语句是一种条件控制语句，按照条件执行不同的代码块

PHP中if语句的格式为：

如果表达式的值为真，就顺序执行语句；否则，就跳过该if语句再往下执行

如果需要执行的语句不只一条，那么可以使用符号{}，来使多个代码组成代码块执行

if…else 条件控制语句

(PHP 4, PHP 5, PHP 7, PHP 8)

if...else语句是一种条件控制语句，它延伸了if语句

如果条件达成，则执行if语句

如果条件不满足，则执行其他语句

PHP 中if...else语句的格式为：

如果表达式的值为真，就执行if的语句1，否则，就执行else的语句2

if…elseif…else 条件控制语句

(PHP 4, PHP 5, PHP 7, PHP 8)

if...elseif...else语句是一种条件控制语句，它延伸了if语句

如果条件达成，则执行if语句

如果条件不满足，则查看elseif的表达式，如果条件达成则执行该语句

否则跳过该语句，直到结束

elseif语句可以有多个，只在第一个elseif的值为真的时候执行

PHP 中if...elseif..else语句的格式为：

先看if，如果表达式为真，则执行语句1，否则向下执行

如果elseif的表达式为真，则执行该语句，并结束整个条件循环，否则判断下一个elseif语句

如果前面的表达式都为假，则执行最后的else语句

switch 条件控制语句

(PHP 4, PHP 5, PHP 7, PHP 8)

switch 是条件控制语句，类似与if语句，但是表现形式不同

switch是把同一个变量或者表达式，与很多不同的值比较，并根据它等于哪个值来执行不同的代码

switch 语句一行接一行地执行（实际上是语句接语句）。开始时没有代码被执行。仅当一个 case 语句中的值和 switch 表达式的值匹配时 PHP 才开始执行语句，直到 switch 的程序段结束或者遇到第一个 break 语句为止。如果不在 case 的语句段最后写上 break 的话，PHP 将继续执行下一个 case 中的语句段

PHP 中switch语句的格式为：

按照顺序，先看条件1，如果条件1与表达式相等，那么执行该case的语句，否则判断下一个case语句

如果case语句中没有和表达式相等的值，那么执行default的语句

在switch语句中，必须在每一个case语句内的结尾包含break语句，用来跳出整个流程语句

在PHP中，case语句不具备划分代码的作用，只起到标记代码的作用

如果不使用break语句退出，在具备条件的case语句内的代码执行之后，会按顺序执行之后的代码

match 条件控制语句

(PHP 8)

match 表达式基于值的一致性进行分支计算。

match表达式和 switch 语句类似， 都有一个表达式主体，可以和多个可选项进行比较。

与 switch 不同点是，它会像三元表达式一样求值。

与 switch 另一个不同点，它的比较是严格比较（ ===）而不是松散比较（==）

match是PHP 8.0.0版本之后出现的语句

如果条件语句和表达式相等，就会返回后面的值

match表达式必须使用分号;结尾

match表达式会返回一个值

for 循环语句

(PHP 4, PHP 5, PHP 7, PHP 8)

PHP 中for语句的格式为：

初始化表达式在循环开始前，无条件求值（并执行）一次

条件表达式在每次循环之前执行，如果结果为真，则执行嵌套的语句，如果结果为假，则终止循环

迭代表达式在每次循环之后执行

每个表达式都可以为空，或者包括逗号分隔的多个表达式

在条件表达式中，所有逗号分隔的表达式都会计算，但是只取最后一个结果

如果条件表达式为空，那么默认为true，会无限循环下去

while 循环语句

(PHP 4, PHP 5, PHP 7, PHP 8)

while 是循环语句，当条件为真时，会循环执行循环体之内的语句，直到条件为假

PHP 中while语句的格式为：

对表达式进行判断，如果条件为真，那么执行循环体内的PHP语句

执行结束后，再返回表达式进行判断，直到表达式的值为假时才会结束循环，执行之后的语句

do…while 循环语句

(PHP 4, PHP 5, PHP 7, PHP 8)

do...while是循环语句，是while语句的另一种形式

两者是区别在于，do...while是先执行代码，然后再对表达式进行判断

PHP 中do...while语句的格式为：

先执行do之后的语句，然后对表达式进行判断

如果表达式的值为真，那么再次循环语句

表达式的真值在每次循环结束后检查

foreach 数组循环语句

(PHP 4, PHP 5, PHP 7, PHP 8)

在PHP中，foreach语句是用于遍历数组，只能够用于数组和对象

同时，数组指针向后移动知道遍历结束

当使用foreach语句时，数组指针将自动被重置，所以不需要手动设置指针位置

declare 语句

(PHP 4, PHP 5, PHP 7, PHP 8)

declare结果用来设定一段代码的执行指令

return 返回语句

(PHP 4, PHP 5, PHP 7, PHP 8)

return语句可以将函数的执行结果返回

如果在一个函数中调用return语句，将立即结束此函数的执行并将它的参数作为函数的值返回

如果在全局范围内使用，则当前脚本文件中止运行

return是语言结构，不是函数

如果被包含的文件使用return则会返回结果

如果被包含的文件使用exit则会退出执行，不再继续执行语句

require 包含语句

(PHP 4, PHP 5, PHP 7, PHP 8)

require是文件包含语句，可以引用外部代码文件

在PHP文件执行之前，解释器会用被引用文件的全部内容替换require语句，然后与require语句之外的其他语句组成新的PHP文件，最后再按新的PHP文件执行程序代码

require语句一般放在源文件的起始位置，需要引用公共资源等

require语句和include语句几乎完全一样，不同的是当被包含文件不存在或者存在错误时

require语句会返回错误并终止程序执行，而include则会返回一个警告但是程序会继续向下执行

include_once 包含语句

(PHP 4, PHP 5, PHP 7, PHP 8)

include_once 语句和 include 语句类似，唯一的区别就是如果包含的文件已经被包含过，就不会再次包含。include_once 可以确保在脚本执行期间同一个文件只被包含一次，以避免函数重定义、变量重新赋值等问题。

require_once 包含语句

(PHP 4, PHP 5, PHP 7, PHP 8)

require_once 语句是 require 语句的延伸，它的功能与 require 语句基本类似，不同的是，在应用 require_once 语句时会先检查要包含的文件是不是已经在该程序中的其他地方被包含过，如果有，则不会再次重复包含该文件。且 include_once 会返回 true

goto 跳转语句

(PHP 5 >= 5.3.0, PHP 7, PHP 8)

goto 操作符可以用来跳转到程序中的另一位置。该目标位置可以用区分大小写的目标名称加上冒号来标记（例如 name: ），而跳转指令是 goto 之后接上目标位置的标记。

PHP 中的 goto 有一定限制，目标位置只能位于同一个文件和作用域，也就是说无法跳出一个函数或类方法，也无法跳入到另一个函数。也无法跳入到任何循环或者 switch 结构中。可以跳出循环或者 switch，通常的用法是用 goto 代替多层的 break。

PHP 中并不鼓励滥用 goto，滥用 goto 会导致程序的可读性严重下降。
