PHP-类型

发表于2021-10-18|更新于2021-10-25|PHPPHP基础
|阅读量:

类型

PHP有十种原始数据类型

四种标量类型：

  • Bool（布尔型）

  • int （整型）

  • float （浮点型，double）

  • string （字符串）

四种复合类型：

  • array（数组）
  • object （对象）
  • callable （可调用）
  • iterable （可迭代）

两种特殊类型：

  • resource （资源）
  • NULL （无类型）

类型声明

只在PHP 7. 4.0以及之后的版本有效

类型声明可以用于声明参数的类型

类型 说明
类/接口/名称 值为指定类和接口的实例化对象
self 值必定是所在方法的类的一个
array 值必须为array
callable 值必定为一个有效的callable，不能用于类属性的类型声明
bool 值必须为一个布尔值
float 值必须为一个浮点数
int 值必须为一个整型数字
string 值必须是字符串
iterable 值必须为array或者instanceof Traversable，可迭代对象
object 值必须为对象
mixed 值可以是任何类型

类型转换

转换语句 转换类型
（Boolean） 转换成布尔型
（string） 转换成字符串
（integer） 转换成整型
（float） 转换成浮点型
（array） 转换成数组
（object） 转换成对象

通过settype()函数转换

类型比较

松散比较：使用两个等号==，只比较值，不比较类型

严格比较：使用三个等号===，比较值和类型

检查数据类型

函数 检测类型
is_bool(x) 检测变量是否为布尔类型
is_string(x) 检测变量是否为字符串
is_float/is_double 检测变量是否为浮点型
is_int/is_integer 检测变量是否为整型
is_null() 检测变量是否为NULL
is_array() 检测变量是否为数组
is_object() 检测变量是否为对象类型
is_numeric() 检测变量是否为数字或者数字组成的字符串

Boolean布尔类型

Boolean 表达了真值，可以为True或者False

两者在PHP中不区分大小写

要明确的将一个值转换成Boolean，用Bool或者Boolean来强制转换

当转换为Boolean时，以下值被认为是False：

  • 布尔值False本身
  • 整型值0以及-0
  • 浮点型值0.0以及-0.0
  • 空字符串，以及字符串"0"
  • 不包括任何元素的数组
  • 特殊类型NULL（包括尚未赋值的变量）
  • 从空标记生成的simpleXML对象

所有其他值都被认为是True

integer整型

整型值int可以用十进制，十六进制，八进制，二进制表示

要使用八进制表达，数字前必须加上0

要使用十六进制表达，数字前必须加上0x

要使用二进制表达，数字前必须加上0b

当给定的数超出了int的范围，将会被解析为float

Float浮点型

通常最大值是1.8e308并且具有14位十进制数字的精度

转换浮点数

如果字符串是数字或前导数字，则它将解析为相应的浮点值，否则将转换为零

对于其他类型，要先转换成int，在转换为float

String字符串

一个字符串 string 就是由一系列的字符组成，其中每个字符等同于一个字节。这意味着 PHP 只能支持 256 的字符集，因此不支持 Unicode 。

转义

符号 含义
\n 换行
\r 回车
\t 水平制表符
\v 垂直制表符
\e Escape
\f 换页
\\ 反斜线
\$ 美元标记
\" 双引号
\' 单引号
\[0-7]{1,3} 符合该正则表达式序列的是一个以八进制方式来表达的字符
\x[0-9A-Fa-f]{1,2} 符合该正则表达式序列的是一个以十六进制方式来表达的字符
\u{[0-9A-Fa-f]+} 匹配正则表达式的字符序列是 unicode 码位， 该码位能作为 UTF-8 的表达方式输出字符串

数字字符串

如果一个PHP String可以被解释为 int 或者 float 类型，则被定义为数字字符串

Array数组

PHP 中的 array 实际上是一个有序映射，映射是一种把value关联到keys的类型

它可以被视为数组，列表（向量），哈希表（映射的实现），字典，集合，堆栈，队列等等

由于 array 的值可以是其它 array 所以树形结构和多维 array 也是允许的

其中的key会被强制转换

  • string中包含十进制int，除非数字前面有+，否则会被转换为int类型，08不会被强制转换
  • floa也会被转换为Int，小数部分会被直接舍去
  • Bool也会被转换成int，True会被转换为1，False会被转换为0
  • NULL会被转换为空字符串
  • Array和object不能被用为键名

如果在数组定义时多个元素都使用相同键名，那么只有最后一个会被使用，其它的元素都会被覆盖

Iterable可迭代对象

Iterable 是PHP 7.1 中引入的一个伪类型，它接受任何array或者实现了Traversable接口的对象

这些类型都能用foreach迭代，也可以和生成器里的yield from一起使用

所有可以被迭代的对象都是可迭代对象，它是一个形式上的说法

Object对象

对象是存储数据和有关如何处理数据的数据类型，是系统中描述客观事物的实体

一个对象由一组属性和对这组属性进行操作的一组服务组成

在PHP中，必须先声明对象

必须声明对象的类，类里面包含了属性和方法

使用关键字Var来声明变量，即创建了类的属性，也叫类的成员属性

通过在类定义中声明函数，即创建了类的方法

Resource资源类型

资源resource是一种特殊变量，保存了到外部资源的引用，资源是通过专门的函数来建立和使用的

由于资源类型变量保存是为打开文件，数据库链接，图形画布等特殊句柄，因此将其他类型转换为资源没有意义

NULL

NULL值表示一个没有变量的值，NULL类型唯一的值就是null

在下列情况下变量被认为是null：

  • 被赋值为null
  • 尚未被赋值
  • 被unset()释放

Callback / Callable类型

回调函数就是通过函数参数传递到其他代码的，某一块可执行代码的引用

也就是将函数作为参数传到另一个函数里面，当那个函数执行结束之后，再执行传过去的那个函数

可以使用callable类型指定回调类型callback

一些函数如call_user_func()或usort()可以接受用户自定义的回调作为参数

回调函数不止可以是简单函数，还可以是对象方法，包括静态类方法
