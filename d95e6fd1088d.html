PHP-运算符

发表于2021-12-09|更新于2021-12-15|编程语言PHPPHP基础
|字数总计:2.5k|阅读时长:8分钟|阅读量:

运算符

运算符是一种告诉编译器执行特定的数学或逻辑操作的符号

运算符是可以通过给出的一个或者多个值（表达式）来产生另一个值（因整个结构成为一个表达式）

运算符的分类

算术运算符

运算符 描述 说明
+$a 标识运算符 将a 转化为 int 或 float
-$a 取反运算符 a 的负值
+ 加法运算符 10 + 20 = 30
- 减法运算符 20 - 10 = 10
* 乘法运算符 2 * 3 = 6
/ 除法运算符 10 / 3 = 3.3333333333333
% 取模，求两个数相除的余数 10 % 6 = 4
** 幂运算符，求一个值的几次幂 2 ** 3 = 8
intdiv() 整除（地板除） intdiv(10,4) = 2

赋值运算符

基本的赋值运算符是 “ = “。它实际上的含义是把右边的表达式赋值给左边的变量

运算符 描述 说明
= 把右边的值赋值给左边 $a = 5 等同于 $a =5
+= 加法赋值运算符 $x += 3 等同于 $x = $x + 3
-= 减法赋值运算符 $x -= 3 等同于 $x = $x - 3
*= 乘法赋值运算符 $x *= 3 等同于 $x = $x * 3
/= 除法赋值运算符 $x /= 3 等同于 $x = $x / 3
%= 取余赋值运算符 $x %= 3 等同于 $x = $x % 3
**= 幂赋值运算符 $x **= 3 等同于 $x = $x ** 3
&= 与赋值运算符 比较两个值的二进制，两个值都为1，则该位结果为1，否则为0
|= 或赋值运算符 比较两个值的二进制，对应的位有一个为1时，结果为1
^= 异或赋值运算符 异或运算符，比较两个值的二进制，对应的位相异时，结果为1
<<= 左移赋值运算符 把左边数值的各二进制位全部左移若干位，右边的数指定移动的位数
>>= 右移赋值运算符 把左边数值的各二进制位全部右移若干位，右边的数指定移动的位数
.= 并置赋值运算符 字符串拼接
??= NULL合并赋值运算符 NULL合并

PHP 支持引用赋值，引用赋值意味着两个变量指向了同一个数据，没有拷贝任何东西

比较运算符

如果关系成立 ，返回True，否则返回False

运算符 描述 说明
== 等于 在类型转换后，比较双方的值是否相等
=== 全等 比较双方的类型和值是否相等
!= 不等 在类型转换后，比较双方的值是否不相等
<> 不等 在类型转换后，比较双方的值是否不相等
!== 不全等 比较双方的类型和值是否不相等
< 小于 比较左边的值是否小于右边
> 大于 比较左边的值是否大于右边
<= 小于等于 比较左边的值是否小于右边，或者等于右边
>= 大于等于 比较左边的值是否大于右边，或者等于右边
<=> 太空船运算符（组合比较符） 同时比较左边的值是否小于，等于，大于右边，返回int值-1,0,1
?? NULL合并运算符 当左边为null，返回右边的值，否则返回左边的值

三元运算符

运算符 描述 说明
$a?$b:$c 三元运算符 通过条件判断来返回不同的值

对于三元运算符$a?$b:$c

如果表达式$a的值为true时，返回$b

如果表达式$a的值为flase时，返回$c

此运算符也可以简写成$a?:$c

如果表达式$a的值为true时，返回$a

如果表达式$a的值为flase时，返回$c

递增/递减运算符

运算符 描述 说明
++$a 前置递增 $a 的值加一，然后返回 $a
$a++ 后置递增 返回 $a，然后将 $a的值加一
–$a 前置递减 $a的值减一，然后返回 $a
$a– 后置递减 返回 $a，然后将 $a的值减一

在运算符处理字符变量时，PHP会使用 Perl 的习惯，而非 C 的

在 Perl 中 $a = 'Z';$a++;，将会把$a变成AA

而在 C 中$a = 'Z';$a++;，将会把$a变成[（Z 的 ASCII 值是90，[ 的 ASCII 值是91）

注意字符变量只能递增，不能递减，并且只支持纯字母（a-z和A-Z）

在上述范围内的其他字符变量，递增或者递减无效，原字符串没有变化

逻辑运算符

运算符 描述 说明
and 逻辑与 如果两边都为true，则返回true
or 逻辑或 如果至少有一个为true，则返回true
xor 逻辑异或 如果仅有一个为true，则返回true
&& 逻辑与 如果两边都为true，则返回true
|| 逻辑或 如果至少有一个为true，则返回true
! 逻辑非 如果变量不为true，则返回true

位运算符

位移在PHP中是数学运算，向任何方向移出去的位都会被丢弃

左移时，右侧以零补充，若符号位被移走意味着正负号不被保留

右移时，右侧移出去的位被丢弃，左侧则以符号位填充

数值表现为32位

运算符 描述 说明
& 按位与 将两边的二进制，将都是1的位设为1，否则为0
| 按位或 将两边的二进制，任何为1的位设为1，否则为0
^ 按位异或 将两边的二进制，其中一边为1，另一边为0的位设为1，否则为0
~ 按位取反 将变量中为1的位设为0，为0的位设为1
<< 左移 将左边$a的位向左移动$b次（每次移动表示乘以2）
>> 右移 将左边$a的位向右移动$b次（每次移动表示除以2）

字符串运算符

运算符 描述 说明
. 并置运算符（连接字符串） ‘hello’.‘123’ = ‘hello123’

数组运算符

当数据类型为数组时，前面的一些运算符，在这里为数组运算符

当两个数组中存在相同的键名，则只有左边数组中的，右边的被忽略

数组中如果具有相同的键名，则比较值

运算符 描述 说明
+ 集合 两边的集合
== 相等 如果两边具有相同的键值对，则为true
=== 全等 如果两边具有相同的键值对，并且顺序和类型都相同，则为true
!= 不等 如果两边不相等，则为true
<> 不等 如果两边不相等，则为true
!== 不全等 如果两百的键值对，并且顺序和类型都不同，则为true

类型运算符

运算符 描述 说明
instanceof 类型运算符 确定变量是否是某一类的实例

instanceof 用于确定一个PHP变量是否属于某一类Class的实例

除了类，也可以使用对象或者字符串变量

如果变量不是对象，则返回false

也可以用来确定一个变量是不是继承自某一个父类的子类的实例

也可以使用逻辑非运算符达到检查对象是否是某个类的实例

执行运算符

PHP支持执行运算符：反引号（``），不是单引号，位于左侧TAB键位的上边

PHP讲尝试反引号中的内容作为shell命令来执行，并将其输出信息返回（赋值给变量而不是丢弃到标准输出）

使用反引号运算符的效果与函数shell_exec()相同

关闭了shell_exec()时反引号运算符是无效的

反引号运算符不能在双引号字符串中使用

运算符 描述 说明
反引号（``） 执行运算符 执行shell命令

错误控制运算符

@运算符，把该运算符放在表达式之前，该表达式可能产生的任何错误信息都会被忽略掉

如果使用了自定义错误信息，则可能返回0

该运算符只对表达式有效，例如，可以把它放在变量，函数和include调用，常量等等

不能放在函数或者类的定义之前，也不能用于条件结构语句例如if和foreach等

该运算符也有导致一些脚本的错误报告失效，可能导致脚本的卡死

运算符 描述 说明
@ 错误控制运算符 忽略表达式可能产生的任何错误信息

运算符优先级

通过运算符的优先级来表示计算顺序

同一行中的运算符具有相同的优先级，结合方向决定了求值顺序

左：从左到右，右：从右到左

例如 1-2-3，如果是左联，就是 (1-2)-3，如果是右联,就是1-(2-3)

在特殊情况下，可以使用括号标明运算顺序，来增加代码的可读性

运算符 描述 结合方向
clone，new clone和new 不适用
** 算术运算符 右
++，–，~，(int)，(float)，(string)，(array)，(object)，(bool)，@ 类型，位，递增递减 不适用
instanceof 类型运算符 左
! 逻辑运算符 不适用
*，/，% 算术运算符 左
+，-，. 算术和字符串运算符 左
<<，>> 位运算符 左
<，<=，>，>= 比较运算符 无
==，!=，===，!==，<>，<=> 比较运算符 无
& 位运算符和引用 左
^ 位运算符 左
| 位运算符 左
&& 逻辑运算符 左
|| 逻辑运算符 左
?? null合并运算符 右
?: 三元运算符 左
=，+=，-=，*=，/=，%=，**=，&=，|=，^=，<<=，>>=，.==，??= 赋值运算符 右
yield from yield from 不适用
yield yield 不适用
print print 不适用
and 逻辑运算符 左
xor 逻辑运算符 左
or 逻辑运算符 左
