运算符(操作符)

运算符的分类

1. 算术运算符

运算符 描述 实例
+ 加法运算符（如果是两个字符串之间进行加法运算，则会进行拼串操作） 12 + 5 = 17
- 减法运算符 12 - 5 = 7
* 乘法运算符（如果将字符串和数字相乘，将字符串重复指定次数） 12 * 5 = 60
/ 除法运算符，运算时结果总会返回一个浮点类型 12 / 5 = 2.4
// 地板除，只会保留计算后的最小整数 12 // 5 = 2
** 幂运算，求一个值的几次幂 12 ** 5 = 248832
% 取模，求两个数相除的余数 12 % 5 = 2

2. 赋值运算符

运算符 描述 说明
= 把右边的值赋值给左边 x = 5 等同于 x =5
+= 加法赋值运算符 x += 3 等同于 x = x + 3
-= 减法赋值运算符 x -= 3 等同于 x = x - 3
*= 乘法赋值运算符 x *= 3 等同于 x = x * 3
/= 除法赋值运算符 x /= 3 等同于 x = x / 3
%= 取余赋值运算符 x %= 3 等同于 x = x % 3
//= 地板除赋值运算符 x //= 3 等同于 x = x // 3
**= 幂赋值运算符 x *= 3 等同于 x = x* 3
&= 与赋值运算符 x &= 3 等同于 x = x & 3
|= 或赋值运算符 x |= 3 等同于 x = x | 3
^= 异或赋值运算符 x ^= 3 等同于 x = x ^ 3
>>= 右移动赋值运算符 x >>= 3 等同于 x = x >> 3
<<= 左移动赋值运算符 x <<= 3 等同于 x = x << 3
:= 表达式内部变量赋值运算符 if (age := 20) > 18: print(“已经成年了！”)

3. 比较运算符

如果关系成立 ，返回True，否则返回False

对字符串比较的是Unicode编码

运算符 描述 实例
> 大于 10 > 20 = Flase
>= 大于或等于 10 >= 20 = Flase
< 小于 10 < 20 = True
<= 小于或等于 10 <= 20 = True
== 比较双方是否相等 10 == 20 = Flase
!= 比较双方是否不相等 10 != 20 = True

4. 逻辑运算符

运算符 描述 实例
and 布尔与，找False，如果没有，则返回右边的值 10 and 20 = 20
or 布尔或，找True，如果没有，则返回右边的值 10 or 20 = 10
not 布尔非，如果为Ture，则返回False；如果为False，则返回Ture； not(10 and 20) = False

5. 条件运算符

语法： 语句1 if 条件表达式 else 语句2
如果判断结果为True，则执行语句1，并返回执行结果
如果判断结果为False，则执行语句2，并返回执行结果

6. 身份运算符

运算符 描述
is 比较两个对象是否是同一个对象，比较的是对象的id
is not 比较两个对象是否不是同一个对象，比较的是对象的id

7. 成员运算符

运算符 描述
in 判断成员是否在指定对象中
not in 判断成员是否不在指定对象中

8. 位运算符

运算符 描述
& 与运算符，比较两个值的二进制，两个值都为1，则该位结果为1，否则为0
| 或运算符，比较两个值的二进制，对应的位有一个为1时，结果为1
^ 异或运算符，比较两个值的二进制，对应的位相异时，结果为1
~ 取反运算符，对数值的每个二进制位取反，即把1变成0，0变成1; 也就是~x = -x -1
<< 左移动运算符，把左边数值的各二进制位全部左移若干位，右边的数指定移动的位数
>> 右移动运算符，把左边数值的各二进制位全部右移若干位，右边的数指定移动的位数

运算符的优先级

运算符 描述
** 指数（最高优先级）
~ + - 按位翻转，一元加号和减号（+@和-@）
* / % // 乘，除，取余，整除
+ - 加法减法
>> << 右，左移运算符
& 位and
^ | 位运算符
<= < > >= 比较运算符
== != 等于运算符
= %= /= //= -= += *= **= 赋值运算符
is is not 身份运算符
in not in 成员运算符
not and or 逻辑运算符
