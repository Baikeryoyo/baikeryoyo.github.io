Python3字符串

字符串可以用单引号，双引号，三引号表示
字符串是序列，可使用所有序列的方法
字符串是不可改变的

字符串类型

​ 在Python中，有两种常用的字符串类型，分别为str和bytes。其中，str表示Unicode字符（ASCII或者其他）；bytes表示二进制数据（包括编码的文本）。这两种类型的字符串不能拼接在一起使用。

​ 通常情况下，str在内存中以Unicode表示，一个字符对应若干个字节。但是如果在网络上传输，或者保存到磁盘上，就需要把str转换为字节类型，即bytes类型。

字符串编码

Python3直接支持Unicode，可以表示世界上任何书面语言的字符。Phthon3的字符默认就是16位的Unicode编码，ASCII码是Unicode编码的子集。

使用内置函数ord()可以把字符串转换成对应的Unicode码；

使用内置函数chr()可以把十进制数字转换成对应的字符；

使用encode()编码，可以把字符串转换为二进制数据(bytes)

使用decode()解码，用于将二进制转换为字符串

Python转义字符

可以使用 \ 作为转义字符，通过转义字符，可以在字符串中使用一些特殊的内容

转义字符 描述
\ (在结尾时) 续航符，连接下一行为同一行语句
\‘ 转义单引号
\\ 转义斜杠
\“ 转义双引号
\t 转义制表符(TAB)
\v 转义纵向制表符(TAB)
\n 转义换行符
\f 转义换页
\b 转义删除键
\r 转义回车
\0 转义为空
\0dd 八进制数，dd代表的字符，如\012代表换行
\xhh 十六进制数，hh代表的字符，如\x0a代表换行
\uxxxx 表示Unicode编码

字符串格式化

符号 描述 实例 结果
%s 格式化字符串（采用str()的显示） print(‘this is %s’ %(‘abcde’)) this is abcde
%r 格式化字符串（采用repr()的显示） print(‘this is %r’ %(‘abcde’)) this is ‘abcde’
%c 格式化字符及其ASCII码 print(‘this is %c’ %34) this is “
%d 格式化整数 print(‘this is %d’ %10.7) this is 10
%o 格式化无符号八进制数 print(‘this is %o’ %8) this is 10
%x 格式化无符号十六进制数 print(‘this is %x’ %10) this is a
%e 用科学计数法格式化浮点数 print(‘this is %e’ %10) this is 1.000000e+01
%f 格式化浮点数；第三位开始会四舍五入 print(‘this is %.3f’ %10.777777) this is 10.778

format()方法占位组合字符串和数字

数字 格式 输出 描述
3.1415926 { : .2f} 3.14 保留小数两位
3.1415926 {: +.2f} 3.14 带符号保留小数两位
2.718 {: .0f} 3 不带小数
5 {: 0>2d} 05 数字补0(填充左边，宽度为2)
5 {: x<4d} 5xxx 数字补x(填充左边，宽度为4)
100000 {: ,} 100,000 以逗号分割数字
0.25 {: .2%} 25.00% 百分比格式
1000000000 {: .2e} 1.00E+09 指数记法
13 {:10d} 13 右对齐（默认，宽度为10）
13 {:<10d} 13 左对齐（宽度为10）
13 {:^10d} 13 居中（宽度为10）
